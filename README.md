rails4-template
===============

Template for Rails 4 - Twitter Bootstrap 3, Haml, Devise, CanCan, Mongoid, Doorkeeper, RSpec, Cucumber, etc.

## Usage ##

```
mkdir myapp
cd myapp
rails new . -m template.rb -T -O
```
